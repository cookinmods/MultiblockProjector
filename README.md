# Multiblock Projector

A universal multiblock projection tool for Minecraft 1.21.1 NeoForge, compatible with all mods.

## Features

- **Universal Compatibility**: Project multiblock structures from any mod
- **Interactive Projections**: Ghost block projections with real-time validation
- **Smart Building Mode**: Visual feedback for correct/incorrect block placement
- **Rotation & Mirroring**: Full control over structure orientation
- **Error Detection**: Instant feedback when incorrect blocks are placed
- **Auto-Completion**: Automatic detection when structures are complete

## Usage

1. **Right-click** the Multiblock Projector to select a structure
2. **Aim and left-click** to place a projection
3. **Right-click while aiming** to rotate the projection
4. **Build blocks** to match the projection
5. **ESC or right-click** to cancel

## Supported Mods

- Immersive Engineering
- Mekanism
- And more! (Compatible with any mod that provides multiblock structures)

## Installation

1. Download the latest release from [Releases](https://github.com/zhintze/MultiblockProjector/releases)
2. Place the `.jar` file in your `mods` folder
3. Start Minecraft with NeoForge 1.21.1

## Development
